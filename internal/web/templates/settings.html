Configuration Reference

Available configuration options

listen: Host and port to bind (e.g., "127.0.0.1:8080")

api_key: API key for authentication (Bearer token)

default_image: Default Docker image for sessions

allowed_images: List of allowed Docker images

db_path: Path to SQLite database file

session_ttl_seconds: Default session lifetime in seconds

defaults.cpu_limit: CPU limit (e.g., 1.0 = 1 core)

defaults.mem_limit_mb: Memory limit in megabytes

defaults.pids_limit: Process limit

defaults.max_exec_timeout_ms: Maximum exec timeout in milliseconds

defaults.network_mode: Network mode ("none", "bridge", "host")

defaults.readonly_rootfs: Whether to make rootfs read-only (true/false)
